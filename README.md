# AI-Powered Smart Traffic Management System

## Overview

This project presents an intelligent traffic management system that uses Computer Vision and Machine Learning to optimize traffic signal timing based on real-time road conditions.

The system uses YOLOv8 for vehicle detection, ByteTrack for vehicle tracking, and an adaptive controller to dynamically adjust traffic light phases.

## Features

- Vehicle Detection
- Vehicle Tracking
- Adaptive Traffic Light Control
- Emergency Vehicle Prioritization
- Queue Length Monitoring
- Congestion Analysis

## Technologies

- Python
- YOLOv8
- ByteTrack
- OpenCV
- Raspberry Pi
